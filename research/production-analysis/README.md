# การวิเคราะห์ผลผลิตโรงปลูกพืช (Production Analysis)

## ภาพรวม
การบันทึกและวิเคราะห์ผลผลิตจริงของฝ่ายโรงปลูกพืช เพื่อใช้เป็นข้อมูล research สำหรับการพัฒนาระบบ IoT และ AI ในการเกษตรแนวตั้ง

## โครงสร้างข้อมูล
- บันทึกผลผลิตแยกตามปี (2024, 2025)
- แยกตามชนิดผัก
- บันทึกรายเดือน (12 เดือน)
- บันทึกรายวันเก็บเกี่ยว
