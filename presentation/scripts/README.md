## 🎯 การใช้งาน

### 📝 Script แต่ละหัวข้อ
- เก็บ script การพูดสำหรับแต่ละหัวข้อ
- ระบุเวลาที่ใช้ในการนำเสนอ
- มี cue สำหรับการเปลี่ยนสไลด์
- รวม Q&A ที่คาดว่าจะถูกถาม

### 📋 Master Script
- รวม script ทั้งหมดเป็นเรื่องเดียว
- ใช้สำหรับการนำเสนอแบบครบชุด
- มี timing และ transition ระหว่างหัวข้อ

## 📋 Template Script

แต่ละ script จะมีโครงสร้าง:

1. **🎯 Opening Hook** (30 วินาที)
2. **❗ Problem Statement** (1-2 นาที)
3. **💡 Solution Overview** (2-3 นาที)
4. **🛠️ Technical Details** (2-3 นาที)
5. **📊 Expected Results** (1-2 นาที)
6. **🎯 Call to Action** (30 วินาที)
7. **❓ Q&A Preparation** (คำถามที่คาดว่าจะถูกถาม)
